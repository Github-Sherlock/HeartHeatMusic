# HeartHeatMusic

## 网易云api微信小程序

##### 根据网易云api制作的小程序 实现在线听歌的功能，实现启动一个微信就能实现网易云听歌的小程序，节约了用户的手机内存，

##### 支持用户输入的歌名/歌手名进行搜索（有版权的音乐不能听）

##### 自动切歌功能上一首播完自动下一首

##### 初学小程序 bug可能有点多 希望大家多多指教。/手动狗头
